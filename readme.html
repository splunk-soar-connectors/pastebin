The 'fetch paste' action downloads the PasteID referenced in the alert email and saves it to the Phantom vault. A widget is also created to display the PasteID, creation time, title author and matching keyword that triggered the alert. A text box in the widget also displays the lines from the paste that matched the keywords.

Setup

  1. Admin -> Apps and click the "+ APP" button to add a new app
  2. Choose the 'pastebin.tgz' file and click install
  3. From the 'Administration' screen click the 'Assets' tab and then the "rest - incident" asset. On the "Ingest Settings" tab copy the values for the ph-auth-token.
  4. Now click back to the "Assets" section and click the "+ ASSET" button.
    • On the "Asset Definition" tab:
      • Asset Name: PasteBin
      • Vendor: PasteBin
      • Product: (should autofill with PasteBin)
    • On the "Asset Settings" tab:
      • Phantom Server IP/Hostname: Should be able to leave this set at 127.0.0.1
      • Phantom REST API token: Paste in the value of 'ph-auth-token' that you copied from the 'rest - incident' asset
  5. Click save.
